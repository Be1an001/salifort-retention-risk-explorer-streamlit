# Data Folder

This folder contains the checked-in HR dataset used by the Streamlit app.

## File

- `hr_capstone_dataset.csv`: Salifort-style HR capstone dataset used for the portfolio project.

The app cleans and standardizes this dataset through `app/utils/load_data.py`. Keeping the CSV in the repo makes the demo reproducible and simple to run locally or on Streamlit Community Cloud.

This is portfolio/demo data, not a live company HR feed.

The app should be read as a responsible analytics demonstration. It is not connected to a real HR system and does not make employment decisions.
