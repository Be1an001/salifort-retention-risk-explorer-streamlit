# Navigator Notes

This folder documents the phased build-out of the PACE Navigator and its advanced review features.

The Navigator is the technical-review layer of the portfolio app. It helps reviewers inspect project truth, drift, retrieval evidence, source previews, audit exports, workflow contracts, and plan-preview guardrails.

## How to Read These Notes

- Start with `demo-readiness-walkthrough.md` for the current demo posture.
- Use `wp*.md` files as implementation history for each navigator work package.
- Use `../streamlit-app-walkthrough.md` for a visitor-friendly page-by-page app walkthrough.

These notes are for technical review. The main app pages and root `README.md` are the best starting point for recruiters and first-time visitors.
