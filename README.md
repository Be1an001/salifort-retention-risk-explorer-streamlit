# Salifort Retention Risk Explorer (Streamlit)

Multipage Streamlit app for operational HR attrition screening, threshold trade-offs, workforce exploration, and explainability using the Salifort Motors case.

## What this app shows

This app presents the public operational version of the Salifort Motors portfolio project as an interactive decision-support tool.

Main themes:
- cost-aware retention screening
- workforce exploration
- workload and tenure risk patterns
- threshold trade-offs
- SHAP-based explainability
- department exposure and manager action view
