# Project Figures

This folder contains stable PNG figures used by the Streamlit pages.

The figures support the portfolio story for:

- exploratory data analysis
- model validation
- threshold tuning
- department exposure
- SHAP explainability
- executive-style project summaries

These images are presentation artifacts. They are useful for the public app, but they are not a substitute for generated model tables in `artifacts/v2/`.

If a generated table and a static figure both exist, the table is usually the better runtime source for app metrics. The figures remain useful for visual storytelling and portfolio review.
